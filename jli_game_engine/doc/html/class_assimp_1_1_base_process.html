Detailed Description

The BaseProcess defines a common interface for all post processing steps. A post processing step is run after a successful import if the caller specified the corresponding flag when calling ReadFile(). Enum aiPostProcessSteps defines which flags are available. After a successful import the Importer iterates over its internal array of processes and calls IsActive() on each process to evaluate if the step should be executed. If the function returns true, the class' Execute() function is called subsequently.
