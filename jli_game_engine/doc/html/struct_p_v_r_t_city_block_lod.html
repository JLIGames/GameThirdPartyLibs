Detailed Description

PVRTCityBlockLod A model tile LOD stores the filename of the POD file and various attributes like the number of (parent) objects it contains. Furthermore it stores two OpenGL buffer object identifiers, one for the vertex and another one for the index data. The last members keep a list of visible nodes which are determined in a seperate visibility update pass.
