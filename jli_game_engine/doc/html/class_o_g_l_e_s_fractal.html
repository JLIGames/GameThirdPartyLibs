InitApplication bool true if no error occurred Code in InitApplication() will be called by PVRShell once per run, before the rendering context is created. Used to initialize variables that are not dependant on it (e.g. external modules, loading meshes, etc.) If the rendering context is lost, InitApplication() will not be called again.

bool OGLESFractal::ReleaseView ( )
virtual

Release the view.

This function can be overloaded by the application. It will be called after the RenderScene() loop, before shutting down the render API. It enables the application to release any memory/resources acquired in InitView().

Returns
true for success, false to exit the application
