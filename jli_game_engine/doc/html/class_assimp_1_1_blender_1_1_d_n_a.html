Detailed Description

Represents the full data structure information for a single BLEND file. This data is extracted from the DNA1 chunk in the file. #DNAParser does the reading and represents currently the only place where DNA is altered.

Fill the converters member with converters for all known data types. The implementation of this method is in BlenderScene.cpp and is machine-generated. Converters are used to quickly handle objects whose exact data type is a runtime-property and not yet known at compile time (consier Object::data).
