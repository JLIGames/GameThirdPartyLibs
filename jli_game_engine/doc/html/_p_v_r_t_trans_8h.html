bool *const  pNeedsZClipping

PVRTBoundingBoxIsVisible pNeedsZClipping pBoundingBox pMatrix TRUE if the object is visible, FALSE if not. Determine if a bounding box is "visible" or not along the Z axis. If the function returns TRUE, the object is visible and should be displayed (check bNeedsZClipping to know if Z Clipping needs to be done). If the function returns FALSE, the object is not visible and thus does not require to be displayed. bNeedsZClipping indicates whether the object needs Z Clipping (i.e. the object is partially visible).

  • *pBoundingBox is a pointer to the bounding box structure.
  • *pMatrix is the World, View & Projection matrices combined.
  • *bNeedsZClipping is TRUE if Z clipping is required.
