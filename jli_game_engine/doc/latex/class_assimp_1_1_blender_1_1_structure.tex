\subsection{Detailed Description}
Represents a data structure in a B\+L\+E\+N\+D file. A \hyperlink{class_assimp_1_1_blender_1_1_structure}{Structure} defines n fields and their locatios and encodings the input stream. Usually, every \hyperlink{class_assimp_1_1_blender_1_1_structure}{Structure} instance pertains to one equally-\/named data structure in the \hyperlink{_blender_scene_8h}{Blender\+Scene.\+h} header. This class defines various utilities to map a binary {\ttfamily blob} read from the file to such a structure instance with meaningful contents.
