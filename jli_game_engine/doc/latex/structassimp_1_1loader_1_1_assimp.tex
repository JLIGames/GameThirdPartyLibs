\subsection{Detailed Description}
Loader class for dynamically loading the \hyperlink{structassimp_1_1loader_1_1_assimp}{Assimp} library.

The library is »reference-\/counted«, meaning that the library is not unloaded on a call to {\ttfamily \hyperlink{structassimp_1_1loader_1_1_assimp_a2380e5e18c3e5a480fafd7a6da93eee5}{unload()}} if there are still other references to it.
