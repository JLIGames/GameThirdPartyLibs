\subsection{Detailed Description}
The \hyperlink{structbt_dbvt_broadphase}{bt\+Dbvt\+Broadphase} implements a broadphase using two dynamic \hyperlink{class_a_a_b_b}{A\+A\+B\+B} bounding volume hierarchies/trees (see \hyperlink{structbt_dbvt}{bt\+Dbvt}). One tree is used for static/non-\/moving objects, and another tree is used for dynamic objects. Objects can move from one tree to the other. This is a very fast broadphase, especially for very dynamic worlds where many objects are moving. Its insert/add and remove of objects is generally faster than the sweep and prune broadphases \hyperlink{classbt_axis_sweep3}{bt\+Axis\+Sweep3} and \hyperlink{classbt32_bit_axis_sweep3}{bt32\+Bit\+Axis\+Sweep3}.
