\subsection{Detailed Description}
Helper class to iterate through all keys in an animation channel.

Missing tracks are interpolated. This is a helper class for \hyperlink{class_assimp_1_1_target_animation_helper}{Target\+Animation\+Helper}, but it can be freely used for other purposes.
