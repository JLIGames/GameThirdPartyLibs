\subsection{Detailed Description}
Class to load M\+D\+L files. 

Several subformats exist\+: 
\begin{DoxyItemize}
\item Quake I 
\item 3\+D Game Studio M\+D\+L3, M\+D\+L4 
\item 3\+D Game Studio M\+D\+L5 
\item 3\+D Game Studio M\+D\+L7 
\item Halflife 2 
\end{DoxyItemize}These formats are partially identical and it would be possible to load them all with a single 1000-\/line function-\/beast. However, it has been split into several code paths to make the code easier to read and maintain.
