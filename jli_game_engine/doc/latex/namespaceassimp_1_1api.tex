\subsection{Detailed Description}
The C-\/style interface to the Open Asset import library.

All functions of the C A\+P\+I have been collected in this module as function pointers, which are set by the dynamic library loader ({\ttfamily \hyperlink{namespaceassimp_1_1loader}{assimp.\+loader}}).

If the call succeeds, the imported data is returned in an {\ttfamily \hyperlink{structai_scene}{ai\+Scene}} structure.
