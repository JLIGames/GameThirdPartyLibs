\subsection{Detailed Description}
Entry point to the jassimp library.

Use \hyperlink{}{import\+File(\+String, Set)} to load a file.

\paragraph*{General Notes and Pitfalls}

Due to the loading via J\+N\+I, strings (for example as returned by the {\ttfamily \hyperlink{namespace_assimp_a0f8daeb06b4e861a2152c33413e6eae1}{get\+Name()}} methods) are not interned. You should therefore compare strings the way it should be done, i.\+e, via {\ttfamily equals()}. Pointer comparison will fail.
