\subsection{Detailed Description}
The \hyperlink{class_assimp_1_1_calc_tangents_process}{Calc\+Tangents\+Process} calculates the tangent and bitangent for any vertex of all meshes. It is expected to be run before the \hyperlink{class_assimp_1_1_join_vertices_process}{Join\+Vertices\+Process} runs because the joining of vertices also considers tangents and bitangents for uniqueness.
