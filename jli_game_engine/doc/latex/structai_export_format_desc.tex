\subsection{Detailed Description}
Describes an file format which \hyperlink{class_assimp}{Assimp} can export to. Use \hyperlink{_assimp_c_export_8cpp_a59cfffbc5b436da8ed8542108102b502}{ai\+Get\+Export\+Format\+Count()} to learn how many export formats the current \hyperlink{class_assimp}{Assimp} build supports and \hyperlink{_assimp_c_export_8cpp_a6896ce9b9851aec848ba11bfc4a525f4}{ai\+Get\+Export\+Format\+Description()} to retrieve a description of an export format option.
