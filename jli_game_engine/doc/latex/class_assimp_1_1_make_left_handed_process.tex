\subsection{Detailed Description}
The \hyperlink{class_assimp_1_1_make_left_handed_process}{Make\+Left\+Handed\+Process} converts all imported data to a left-\/handed coordinate system. 

This implies a mirroring of the Z axis of the coordinate system. But to keep transformation matrices free from reflections we shift the reflection to other places. We mirror the meshes and adapt the rotations.

\begin{DoxyNote}{Note}
R\+H-\/\+L\+H and L\+H-\/\+R\+H is the same, so this class can be used for both 
\end{DoxyNote}
