\subsection{Detailed Description}
Post-\/processing filter to split large meshes into sub-\/meshes

Applied B\+E\+F\+O\+R\+E the Join\+Vertices-\/\+Step occurs. Returns N\+O\+N-\/\+U\+N\+I\+Q\+U\+E vertices, splits by triangle number.
