\subsection{Detailed Description}
Irr\+Mesh importer class.

Irr\+Mesh is the native file format of the Irrlight engine and its editor irr\+Edit. As Irr\+Edit itself is capable of importing quite many file formats, it might be a good file format for data exchange.
