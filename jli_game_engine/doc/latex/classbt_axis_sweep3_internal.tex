It uses quantized integers to represent the begin and end points for each of the 3 axis. Dont use this class directly, use \hyperlink{classbt_axis_sweep3}{bt\+Axis\+Sweep3} or \hyperlink{classbt32_bit_axis_sweep3}{bt32\+Bit\+Axis\+Sweep3} instead.
