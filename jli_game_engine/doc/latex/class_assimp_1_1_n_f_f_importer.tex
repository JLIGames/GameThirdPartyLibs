\subsection{Detailed Description}
N\+F\+F (Neutral \hyperlink{class_file}{File} Format) \hyperlink{class_assimp_1_1_importer}{Importer} class.

The class implements both Eric Haynes N\+F\+F format and Sense8's N\+F\+F (N\+F\+F2) format. Both are quite different and the loading code is somewhat dirty at the moment. Sense8 should be moved to a separate loader.
