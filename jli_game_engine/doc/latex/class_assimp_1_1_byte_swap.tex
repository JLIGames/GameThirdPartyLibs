\subsection{Detailed Description}
Defines some useful byte order swap routines.

This is required to read big-\/endian model formats on little-\/endian machines, and vice versa. Direct use of this class is D\+E\+P\+R\+E\+C\+A\+T\+E\+D. Use \#\+Stream\+Reader instead.
